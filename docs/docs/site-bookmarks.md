# 书签管理

> 独立于账号的书签收藏功能，可保存常用的站点地址（控制台/文档/管理页面），无需创建完整账号即可快速访问。

## 功能概述

- **轻量级收藏**：仅保存 `name + URL`，无需绑定账号信息
- **标签分类**：支持为书签添加标签，方便筛选和管理
- **备注说明**：支持添加备注信息，记录书签用途
- **置顶与排序**：支持置顶和拖拽排序，优先显示常用书签
- **快速操作**：一键打开书签、复制 URL、编辑、删除
- **全局搜索**：支持按名称、URL、备注、标签搜索

## 与账号的区别

| 特性 | 账号 | 书签 |
|------|------|------|
| 绑定站点信息 | ✅ 完整账号数据 | ❌ 仅 name + URL |
| 余额/用量 | ✅ 支持 | ❌ 不支持 |
| API 密钥管理 | ✅ 支持 | ❌ 不支持 |
| 自动签到 | ✅ 支持 | ❌ 不支持 |
| 自动刷新 | ✅ 支持 | ❌ 不支持 |
| 模型列表 | ✅ 支持 | ❌ 不支持 |

书签**不会**出现在密钥管理、模型管理、余额汇总、签到自动化等账号专属流程中。

## 使用场景

- 快速访问常用的管理后台地址
- 保存文档/控制台/监控页面的快捷链接
- 记录需要频繁访问但不需要完整账号管理的站点

## 使用方法

### 添加书签

1. 打开插件弹窗，点击 **「账号/书签」** 切换到书签 tab
2. 点击 **「添加书签」** 按钮
3. 填写书签信息：
   - **名称**：书签显示名称（必填）
   - **URL**：目标站点地址（必填）
   - **备注**：可选的说明信息
   - **标签**：选择或创建标签
4. 点击「保存」

### 管理书签

- **打开**：点击书签名称或 URL 即可在新标签页打开
- **复制 URL**：点击复制图标将 URL 复制到剪贴板
- **编辑**：点击编辑图标修改书签信息
- **删除**：点击删除图标，确认后删除书签
- **置顶**：点击置顶图标将书签固定在列表顶部
- **拖拽排序**：拖动书签行调整顺序

### 搜索与筛选

- 在搜索框中输入关键词，可搜索名称、URL、备注、标签
- 点击标签可筛选对应标签的书签

## 数据同步

书签数据会包含在：
- JSON 导入/导出备份中
- WebDAV 自动同步中
- 分享快照中（可选是否包含）

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 书签点击无反应 | 检查浏览器是否允许打开新标签页 |
| 搜索不到结果 | 确认搜索词是否匹配名称、URL、备注或标签 |
| 无法添加标签 | 先在账号管理中创建标签，书签即可复用 |
