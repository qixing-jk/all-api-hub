# 更新ログ

このページでは、一般ユーザー向けの主な更新点（機能変更 / 体験改善 / 問題修正）を記録しています。完全な履歴バージョンとより詳細な技術変更については、[GitHub Releases](https://github.com/qixing-jk/all-api-hub/releases) を参照してください。

::: tip 新規ユーザー向けガイド
- **現在のバージョンを確認する方法**：拡張機能のポップアップを開くと、タイトルバーにバージョン番号が表示されます。設定ページでも確認できます。
- **このページが自動的に開かないようにする方法**：「設定 → 一般 → 更新ログ」で、「更新後に更新ログを自動的に開く」を制御できます。
- **異常のトラブルシューティング**：「設定 → 一般 → ログ」でコンソールログを有効にし、再現手順を添えて [Issues](https://github.com/qixing-jk/all-api-hub/issues) に報告してください。
:::

## 3.36.0
- **新機能:**
  - アカウント管理：無効化されたアカウントの「ワンクリック更新」機能を追加しました。アカウントが再度利用可能になったかを素早く確認できます。
  - キー管理：複数アカウントのキーフィルタリングとグループ選択をサポートし、キーの整理がより効率的になりました。
  - キー管理：キーをホストサイトチャネルに一括エクスポートする際に、モデルリストを直接編集できるようになりました。エクスポート後に手動で変更する手間を減らしました。
  - チャネル移行：`AxonHub` および `Claude Code Hub` サイトへの移行サポートを追加しました。サイトを跨いだ同期がより簡単になりました。
  - インターフェース改善：オプションページヘッダーにフィードバックのクイック入口を追加しました。クリックすると GitHub の Issue 提出ページに素早く移動できます。
- **体験改善:**
  - ローカライゼーション：地域ラベル、テレメトリモードラベルなどの細部のローカライズ翻訳を改善しました。
  - インターフェース表示：ダイアログ結果リストとプレビューリストの最大表示高さを引き上げ、長いリストを表示する際に、よりクリアになりました。
- **問題修正:**
  - インターフェース操作：チェックボックスの半選択（不確定）状態が、一括選択プロセス中に正しく保持されない問題を修正しました。

**場所のヒント:**
- 無効化されたアカウントの更新：「設定 → アカウント管理」で、無効化されたアカウントのエントリに対して行います。
- キーの一括エクスポートとフィルタリング：「設定 → キー管理」にあります。
- フィードバック入口：オプションページのヘッダー領域にあります。

## 3.35.0
- **新機能:**
  - ホストサイト：`AxonHub` 統合設定を追加しました。拡張機能内で `Base URL`、管理者メールアドレス、パスワードを設定し、接続を検証した後、AxonHub チャネルを直接読み取り、作成、編集、削除できます。既存のアカウント / キーを AxonHub チャネルとしてインポートすることも可能です。また、チャネル移行でソースまたはターゲットとして使用できます。
  - ホストサイト：`Claude Code Hub` 統合設定を追加しました。`Base URL` と管理者トークンを設定し、`Provider` リストを直接管理できます。既存のアカウント / キーを Claude Code Hub プロバイダーとしてインポートすることもサポートし、複数のバックエンド間での重複入力を減らします。
- **問題修正:**
  - インポート/エクスポート：設定を含むバックアップをインポートした後、設定状態が即座に更新され、バックアップ内のインターフェース言語が適用されるようになりました。これにより、インポート完了後も古い設定や古い言語が表示され、手動でページを再オープンする必要がある状況が減少しました。
  - アカウント管理：`現在のログインアカウント優先`、`一致する開いているタブ` の 2 つのソート / マッチングルールの説明と実際の動作が一致しました。完全な URL、タブタイトル、または現在のページ関連コンテンツを使用してアカウントを検索する際に、関連アカウントがより正確に認識され、上位に表示されるようになりました。

::: tip 使用範囲の説明
`Claude Code Hub` の現在のバージョンでは、`チャネル移行`、`モデル同期`、`モデルリダイレクト`、および完全なキー表示はサポートされていません。`AxonHub` は現在、`モデル同期` および `モデルリダイレクト` などの New API 専用操作をサポートしていません。
:::

**場所のヒント:**
- `AxonHub` / `Claude Code Hub` 統合設定と検証：「設定 → 基本設定 → 自社サイト管理」にあります。
- `AxonHub` チャネル管理、`Claude Code Hub` プロバイダー管理、およびターゲットバックエンドへのインポート：「設定 → チャネル管理」および関連するインポートプロセスにあります。
- 設定バックアップのインポート：「設定 → インポート/エクスポート」にあります。
- アカウントソート優先度とアカウント検索：「設定 → アカウント管理」にあります。
