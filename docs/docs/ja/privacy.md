# All API Hub プライバシーポリシー

**最終更新日：** 2025年12月30日

## 概要

All API Hub は、AI APIアグリゲーションサービスのユーザーアカウント情報の管理を支援するブラウザ拡張機能です。本プライバシーポリシーは、お客様のデータの取り扱いについて説明します。

---

## データ収集と保存

## データ収集

本拡張機能は、**いかなる個人データも収集せず、当社のサーバーに送信しません**。具体的には以下の通りです。

* ユーザーデータを収集したり、いかなるサーバーにも送信したりしません
* いかなる分析ツールや追跡ツールも使用しません
* 閲覧履歴を記録しません
* 個人を特定できる情報を保存しません

---

### データの保存方法

すべてのデータは、**ローカルデバイスにのみ保存されます**。これには以下が含まれます。

* ブラウザのローカルストレージ
* 当社が運営するいかなる外部サーバーにもデータは送信されません
* データはお客様のデバイスにのみ保持され、ブラウザの同期機能を有効にした場合にのみ、ブラウザに搭載されている同期メカニズムを通じて同期されます

---

### オプションのクラウド同期機能

本拡張機能は、**オプションの** WebDAV同期機能を提供します。

* お客様は、アカウントデータを自身のWebDAVサーバーに同期することを選択できます
* この機能は**デフォルトで無効**です
* 有効にした場合、データは指定されたWebDAVサーバーに安全な接続（HTTPS）を介してのみ送信されます
* 当社は、お客様のWebDAVサーバーや関連する認証情報にアクセスすることはできません

---

## データの利用方法

本拡張機能がアクセスまたは保存するデータは、お客様が本拡張機能の機能を利用するためにのみ使用されます。
これらのデータは、分析、広告、ユーザープロファイリング、追跡などの目的には使用されません。

---

## サードパーティサービス

本拡張機能は、**いかなるサードパーティの分析、広告、またはデータ収集サービスも統合していません**。

お客様が**明示的に設定した場合にのみ**、本拡張機能は以下のサードパーティサービスと通信します。これには以下が含まれますが、これらに限定されません。

* お客様が追加したAI APIアグリゲーションプラットフォーム（例：one-api、new-apiなど）
* お客様が設定したWebDAVサーバー（同期機能を有効にした場合など）

これらのサービスとのすべての通信は、お客様によって能動的にトリガーされ、対応する機能を実装するためにのみ使用されます。
当社は、これらのサービスに送信されるデータを一切受信、監視、または保存しません。

---

## プライバシーポリシーの変更

当社は、本プライバシーポリシーを不定期に更新する場合があります。いかなる変更も「最終更新日」に反映されます。
ポリシー更新後に本拡張機能を引き続き使用することは、更新されたプライバシーポリシーに同意したものとみなされます。
