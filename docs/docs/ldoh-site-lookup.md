# LDOH 站点查询

> 查询 LDOH 站点信息，帮助发现和添加新的中转站。

## 功能概述

- **站点发现**：通过 LDOH 服务查询可用站点
- **站点信息**：查看站点地址、状态、配置信息
- **一键添加**：从查询结果直接添加站点到插件
- **历史记录**：保存查询历史，方便快速访问

## 使用方法

### 查询站点

1. 进入 **设置 → LDOH 站点查询**
2. 在搜索框中输入关键词（如站点名称、域名）
3. 点击「搜索」或按回车键
4. 查看搜索结果列表

### 添加站点

1. 在搜索结果中找到目标站点
2. 点击「添加」按钮
3. 确认站点信息
4. 站点将添加到账号管理中

### 查看详情

1. 点击搜索结果中的站点
2. 查看详细信息：地址、状态、配置等
3. 可选择添加或返回列表

## 数据来源

LDOH 站点查询的数据来源于公共 LDOH 服务，可能包含：
- 用户分享的站点
- 公开的中转站列表
- 社区验证的可用站点

## 使用建议

- 首次使用建议先浏览热门站点
- 添加前确认站点可正常访问
- 定期检查已添加站点的可用性

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 搜索无结果 | 尝试不同的关键词或检查网络连接 |
| 添加失败 | 确认站点可访问，或手动添加账号 |
| 站点不可用 | 站点可能已下线，尝试其他站点 |
