# 分享快照

> 将当前配置（账号、书签、设置等）导出为快照文件，分享给他人导入使用，支持加密保护敏感信息。

## 功能概述

- **完整配置导出**：导出账号、书签、设置等完整配置
- **敏感信息控制**：可选择是否包含 API Key 等敏感信息
- **加密保护**：支持设置密码保护快照文件
- **增量更新**：可选择导出完整配置或仅导出部分配置
- **导入兼容**：支持导入他人分享的快照

## 使用场景

- **团队共享**：将配置分享给团队成员，统一管理多账号
- **备份恢复**：导出配置作为备份，换设备时可快速恢复
- **问题排查**：导出配置用于排查问题或反馈给开发者
- **模板分享**：分享标准配置模板供他人使用

## 使用方法

### 导出快照

1. 进入 **设置 → 数据与备份**
2. 点击 **「分享快照」**
3. 选择导出内容：
   - **账号配置**：包含站点账号信息
   - **书签**：包含书签数据
   - **API 凭证档案**：包含独立 API 凭证
   - **用户设置**：包含插件设置
4. 选择是否包含敏感信息：
   - **包含 API Key**：完整导出
   - **不包含 API Key**：仅导出配置结构
5. （可选）设置密码保护
6. 点击「生成快照」
7. 下载生成的快照文件（`.json` 格式）

### 导入快照

1. 进入 **设置 → 数据与备份**
2. 点击 **「导入快照」**
3. 选择快照文件
4. 如果快照有密码保护，输入密码
5. 预览导入内容
6. 确认导入

## 快照内容

快照文件为 JSON 格式，包含以下部分：

| 字段 | 说明 | 可选敏感 |
|------|------|----------|
| version | 快照格式版本 | - |
| exportedAt | 导出时间 | - |
| accounts | 账号配置 | ✅ API Key |
| bookmarks | 书签数据 | - |
| apiCredentialProfiles | API 凭证档案 | ✅ API Key |
| userPreferences | 用户设置 | - |

## 安全建议

- 分享包含敏感信息的快照时，建议设置密码
- 不在公开场合分享包含 API Key 的快照
- 收到他人快照时，检查内容后再导入

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 导入失败 | 检查快照文件格式是否正确 |
| 密码错误 | 确认密码输入正确，若忘记密码无法恢复 |
| 部分数据丢失 | 检查快照文件是否完整 |
