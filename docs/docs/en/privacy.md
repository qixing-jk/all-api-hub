# All API Hub Privacy Policy

**Last Updated:** December 30, 2025

## Overview

All API Hub is a browser extension designed to help users manage account information for AI API aggregation services. This Privacy Policy explains how we handle your data.

---

## Data Collection and Storage

## Data Collection

This extension **does not collect or transmit any personal data to our servers**. Specifically:

*   Does not collect or transmit user data to any server
*   Does not use any analytics or tracking tools
*   Does not record browsing history
*   Does not store any personally identifiable information

---

### How Data is Stored

All data is **stored exclusively on your local device**, including:

*   Your browser's local storage
*   No data is transmitted to any external servers operated by us
*   Data remains solely on your device and is only synchronized via your browser's built-in synchronization mechanism if you enable the browser's sync feature

---

### Optional Cloud Sync Feature

This extension offers an **optional** WebDAV synchronization feature:

*   You can choose to synchronize your account data to your own WebDAV server
*   This feature is **off by default**
*   When enabled, data is only transmitted to your specified WebDAV server via a secure connection (HTTPS)
*   We cannot access your WebDAV server or related credentials

---

## How Data is Used

Any data accessed or stored by this extension is used solely to enable the functionality you use within the extension.
This data is not used for analytics, advertising, user profiling, or tracking purposes.

---

## Third-Party Services

This extension **does not integrate any third-party analytics, advertising, or data collection services**.

This extension will only communicate with the following third-party services **if explicitly configured by you**, including but not limited to:

*   AI API aggregation platforms you add (e.g., one-api, New API, etc.)
*   Your configured WebDAV server (e.g., if sync functionality is enabled)

All communication with these services is actively initiated by you and solely for the purpose of enabling the corresponding functionality.
We do not receive, monitor, or store any data transmitted to these services.

---

## Changes to the Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last Updated" date.
Your continued use of this extension after a policy update signifies your agreement to the updated Privacy Policy.
