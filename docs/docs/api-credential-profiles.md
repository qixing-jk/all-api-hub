# API 凭证档案

> 独立于站点账号保存 `baseUrl + API Key` 组合，可脱离站点单独管理第三方 API 凭证，支持标签筛选和复用。

## 功能概述

- **独立凭证管理**：不依赖站点账号，独立保存 API 凭证
- **标签分类**：支持为凭证添加标签，方便筛选
- **模型查看**：可查看凭证对应的可用模型列表
- **接口验证**：支持测试 API 可用性
- **状态统计**：查看凭证的使用状态和健康度
- **批量操作**：支持批量导出、删除凭证

## 使用场景

- 管理未通过站点账号添加的独立 API 凭证
- 记录第三方 API 服务（如 OpenAI、Anthropic、Google 等）的直接调用凭证
- 按标签分类管理不同用途的凭证
- 快速复用凭证到模型查看、接口验证等场景

## 与账号的区别

| 特性 | 账号 | API 凭证档案 |
|------|------|--------------|
| 站点绑定 | ✅ 绑定到特定站点 | ❌ 独立凭证 |
| 余额/用量 | ✅ 从站点获取 | ❌ 需手动更新 |
| 自动刷新 | ✅ 支持 | ❌ 不支持 |
| 站点类型检测 | ✅ 自动识别 | ❌ 手动填写 |

## 使用方法

### 添加凭证

1. 进入 **密钥管理** 页面
2. 切换到 **「API 凭证档案」** tab
3. 点击 **「添加凭证」**
4. 填写凭证信息：
   - **名称**：凭证显示名称
   - **Base URL**：API 基础地址（如 `https://api.openai.com/v1`）
   - **API Key**：API 密钥
   - **标签**：选择或创建标签（可选）
5. 点击「保存」

### 管理凭证

- **查看模型**：点击凭证查看可用模型列表
- **验证接口**：点击测试按钮验证凭证可用性
- **编辑**：修改凭证信息
- **删除**：删除凭证（需确认）
- **复制**：复制 Base URL 或 API Key

### 筛选与搜索

- 通过标签筛选凭证
- 搜索凭证名称

## 数据同步

API 凭证档案会包含在：
- JSON 导入/导出备份中
- WebDAV 自动同步中
- 分享快照中（可选是否包含敏感信息）

## 安全建议

- 敏感凭证建议启用分享快照加密功能
- 定期检查凭证有效性，及时更新或删除失效凭证
- 不要在公开场合分享凭证信息

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 验证失败 | 检查 Base URL 和 API Key 是否正确 |
| 模型列表为空 | 凭证可能无权访问模型列表接口 |
| 无法添加标签 | 先在账号管理中创建标签 |
